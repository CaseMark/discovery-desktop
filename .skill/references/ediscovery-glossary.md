# E-Discovery Glossary

Legal terminology relevant to discovery-desktop.

## Core Concepts

| Term | Definition |
|------|------------|
| **Discovery** | Pre-trial phase where parties exchange relevant information |
| **E-Discovery** | Discovery of electronically stored information (ESI) |
| **ESI** | Any digital information: emails, documents, databases, media |

## Document Types

| Type | Description |
|------|-------------|
| **Pleadings** | Court filings (complaints, answers, motions) |
| **Interrogatories** | Written questions requiring sworn answers |
| **RFP** | Request for Production - formal document requests |
| **RFA** | Request for Admission - requests to admit/deny facts |
| **Depositions** | Sworn out-of-court testimony transcripts |
| **Exhibits** | Documents/items introduced as evidence |

## Process Stages

1. **Identification** - Locate custodians and data sources
2. **Preservation** - Legal hold to prevent spoliation
3. **Collection** - Gather ESI maintaining chain of custody
4. **Processing** - Dedupe, filter, OCR, extract metadata
5. **Review** - Assess relevance, privilege, responsiveness
6. **Production** - Deliver documents with Bates numbers

## Key Legal Concepts

| Concept | Meaning |
|---------|---------|
| **Privilege** | Protected communications (attorney-client, work product) |
| **Proportionality** | Discovery burden must be proportional to case needs |
| **Relevance** | Information relating to claims or defenses |
| **Spoliation** | Destruction/alteration of evidence (serious consequences) |

## Search Terms

| Term | Definition |
|------|------------|
| **Keyword Search** | Boolean/proximity term matching |
| **Semantic Search** | AI-powered meaning/concept search |
| **Relevance Score** | Algorithmic ranking of document relevance (0-1) |
| **TAR** | Technology Assisted Review - ML-based relevance prediction |
